How to Build

Requirement

Microsoft Visual Studio 2008

Boost library 1.350 from http://boost.org or http://boostpro.com

 

cf) standard C++ 와 boost library 를 이용해서 대부분의 코드가 작성되었습니다. 클라이언트는 visual 부분에서 MFC 가 사용되었습니다.  리눅스 코딩 경험이 없어서 리눅스 지원은 하지 않으나 core library 는 대부분 standard C++ 로 작성되었습니다 .

 

cf) Medicalphoto 에는 다음과 같은 기술적인 특징이 있습니다.

1) Full Unicode support (UTF-8 <-> UTF16-LE(windows) <-> local locale <-> UTF-8 )

2) 자체 캐쉬기능을 이용한 이미지 전송.

3) boost asio 를 이용한 네트워크 - 클라이언트 구조

4) 멀티 모니터 지원

5) www 을 이용한 프로그램 자동 업데이트

6) 대부분의 클라이언트는 boost::thread 를 이용해서 지연이 없도록 만들었습니다.

7) installer 지원 - update 시 silent installer 와 표준 installer

8) Sqlite 를 이용한 내장 database

9) exim 을 이용한 사진의 EXIF 정보 분석

10) FTP 서버 지원 (cftpserver.c)

11) 윈도우 폴더내의 파일의 변화를 탐지해서 사진이 카피되면 자동으로 업로드 하는 기술. Cdirectorywatch
