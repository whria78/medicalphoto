1. I have existing patient photo records. Can I use past photos in MedicalPhoto?

Suppose your existing photos are organized as follows:

d:/old_photo ‚Üí d:/old_photo/Smith/2011-1-1/IMG_1234.jpg

Follow these steps to install MedicalPhoto while keeping past photos:

  • Create a folder named d:/clinical_photos.
  • Move d:/old_photo to d:/clinical_photos/old_photo.
  • Install MedicalPhoto.
  • Set the installation folder as d:/clinical_photos.
  • Search for Smith in MedicalPhoto, and it will find past photos, as folder names are included in the search.

The files will be stored as follows:

2. My hospital has multiple locations. Can I manage photos from different locations together?

If you configure your server's router to forward TCP port 9999 to the server computer, external access will be possible.

However, sharing photos over the internet is not recommended due to security concerns.

3. Is there a way to quickly select multiple patient photos at once while organizing them?

Hold down the CTRL key while selecting photos in MedicalPhoto:

  • Similar filenames or metadata-based photos will be selected first.
  • Photos taken within approximately 10 seconds of the selected photo will also be chosen.

4. What happens to my stored photos if MedicalPhoto stops working in the future?

Even if MedicalPhoto stops working, your photos remain stored in the hard drive as follows:

For example, if MedicalPhoto displays a file as:

clinical_photos/2012/aaa

It is actually stored as:

d:/clinical_photos/2012/aaa.jpg

To find the storage location:

Start ‚Üí Programs ‚Üí MedicalPhoto ‚Üí s_config.txt

5. I want to delete multiple photos at once.

To delete multiple photos quickly:

  • Open the folder where MedicalPhoto stores images (e.g., d:/clinical_photos).
  • Delete the unwanted files directly in Windows File Explorer.
  • MedicalPhoto will automatically detect the deleted files and update the database.

6. How can I back up MedicalPhoto data?

If d:/clinical_photos is the storage folder, the database file is:

d:/clinical_photos/clinical_photo.dat

To back up data: Save this file along with your photos. To restore, copy it back to the same location.
